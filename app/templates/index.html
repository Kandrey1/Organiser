Модуль - Погода:

Данный модуль показывает погоду на две недели вперед и архив погоды.

Прогноз погоды на две недели вперед осуществлен с помощью парсинга страницы с сайта gismeteo.

Архив погоды берется из БД, если в архиве отсутствуют необходимые данные, то с сайта gismeteo парситься необходимый архив погоды и сохраняется в БД .

Так же сделан API REST для запроса прогноза погоды и архива погоды (Прогноз можно получать как по url, так и с помощью отправка json).
